# OOP Fundamentals in Java

This repository contains exercises and examples developed during
Object-Oriented Programming classes, using Java as the main language.

## Topics covered
- Classes and objects
- Encapsulation
- Inheritance
- Method overloading and overriding
- Basic polymorphism
